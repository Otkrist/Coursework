In this assignment we look at effects of mixing color channels and varying light sources. I used a set up with 3 light sources, All three sources were of identical make. The three images I obtained come below.
Light Source in Right Light Source in Left Light Source in Front


We can mix color channels from source (a), source (b), source (c) in various innovative ways. Here are some of interesting solutions that I obtained.

Image Color Channels are Added. A=(A1.r,A2.g,A3.b) Image Color Channels are Added and normalised. A=(A1.r2,A2.g2,A3.b2)/(A1.r2+A2.g2+A3.b2)
Only Maximum Color Channel is taken. This picture can be used to identify which light source is most active in which region. Only Minumum Color Channel is taken. This picture depicts which light source is least active in which region.
The color channels are made cosine functions of their respective functions. The reason behind this was to induce inverse variation of color intensity preserving intenisity>1 bounds. Max and Min color channels are added.


I came up with a mathematical model in which we treat color information of each of 3 input images as a column vector in a matrix of 3 x 3. And I obtained the color channels of the output image by performing matrix operations on this matrix and multiplying it with a vector. The results with the operand applied come below.
Image Output By mixing only one color channel from each image. (A=Diagonal) Image output by using all color channels to construct A matrix. Operand
Simple multiplication by [1;1;1]
Multiplication by diamond shaped [[0;0.33;0],[0.33;0;0.33],[0;0.33;0]]
(A*A+2*A+I)[1;1;1]
(A*A-2*A+I)[1;1;1]
(A*A*A-3*A*A+3*A-I)[1;1;1]
